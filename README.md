# ESG Momentum Engine

**PolyFinTech100 2026 · Sponsored by CGS International**

Three-layer ESG investment intelligence dashboard for brown industry ASEAN-listed companies.

## Deploy to Vercel

```bash
npm install && npm run build
vercel --prod
```

Or connect this repo to [vercel.com](https://vercel.com) for automatic deploys on push.

## Local dev

```bash
npm install
npm run dev
```

## Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · Recharts · Framer Motion
